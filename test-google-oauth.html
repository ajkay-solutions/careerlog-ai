Error Message Decoded:

invalid_client: The OAuth client was not found

Solution:

You need to either:

  1. Create a new OAuth 2.0 Client ID in Google Cloud Console
  2. Or find the correct client ID that exists in your Google Cloud project

Steps to Create New OAuth Client:

  1. Go to Google Cloud Console - Credentials
  2. Click "Create Credentials" > "OAuth client ID"
  3. Choose "Web application"
  4. Add authorized redirect URIs:
    • http://localhost:3004/auth/google/callback
    • https://worklog-ai-backend.onrender.com/auth/google/callback
    • https://worklog.ajkaysolutions.com/auth/google/callback
  5. Copy the new Client ID and Client Secret
  6. Update the .env file with the new credentials
